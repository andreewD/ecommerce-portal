# ECOMMERCE PORTAL

## DEPENDENCIES

- NodeJS 18.2.0
- NPM 6.14.5
- React 18.2.0
- React Credit Cards 2 1.0.1
- React DOM 18.2.0
- Typescript 4.9.5
- Axios 1.5.1

## INSTALLATION

- Clone the repository
- Run `npm install` to install dependencies
- Run `npm run start` to start the application
- Open `http://localhost:3000` in your browser
